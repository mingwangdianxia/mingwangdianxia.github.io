汇总java常用的小技巧，小知识

快捷键

  • ctrl + shift +F8：显示所有断点
  • F11：打标签

关键字

default

在接口的方法中用这个关键字可以设置默认的实现，目的是在实现接口时，类可以不重写被default修饰的方法

instanceof

严格来说是java的一个双目运算符，用来测试一个对象是否为一个类的实例, 下面实例中obj是一个对象，Class是标识一个类或者一个接口，当obj是Class的对象（或是Class的子类 | 间接子类 | 接口的实现）时，返回true，否则是false

obj instanceof Class
技巧

总结：

1、obj是由Class直接创建的

2、obj是由继承了Class的类创建的

3、obj是由继承了Class的类的下一个继承类创建的

4、Class是接口，obj由实现了Class的类创建的

注意：

1、obj必须是引用类型，不能是基本类型

2、obj为null时，返回false

3、编辑器会检查obj是否能转换成右边Class类型，如果不能转换则直接报错

常用命令

  • javac：编译代码，将.java文件编译成.class文件
  • java：解释代码，运行class文件
  • javap：反编译：
  • javadoc：生成文档：生成文档

项目运行、部署

  1. 借助idea工具运行
run as xxx | debug as xxx
  1. 普通运行部署
java -jar xxx.jar
  1. linux 后台运行：
nohup java -jar xxx.jar > out.txt &

# &: 后台运行
# nohup : 不挂断的运行命令
# > : 将日志文件写到out.txt文件中
  1. windows 后台运行 编写bat文件，在文件中输入下面代码，然后关闭双击这个文件
@echo off
start java -jar xxx.jar
exit

5、指定配置

java -jar xxx.jar --spring.profiles.active=prod
java -jar -Dspring.profiles.active=test xxx.jar
ENTRYPOINT ["java","-jar","-Dspring.profiles.active=prod","xxx.jar"]

运算符

&: 按位与

|: 按位或

&&: 逻辑与

||: 逻辑或

^: 异或运算（不同为1，相同为0）

<<: 左移 m « n : m*（2的n次方）

>> 右移 m»n ：m/(2的n次方)

~: 取反运算符

>>> 无符号右移运算
