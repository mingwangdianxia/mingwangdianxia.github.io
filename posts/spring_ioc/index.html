控制反转

  • 控制反转是一种思想
  • 为了降低程序耦合性，提高程序扩展力，达到OCP原则，DIP原则
  • 反转的是对象的创建权力
  • 实现：依赖注入

依赖注入

Spring通过依赖注入的方式来实现Bean管理的

  • 依赖指的是对象和对象之间的关联关系
  • 注入指的是一种数据传递行为
  • 注入方式：set、构造器

set注入

想要Spring调用对应的set方法，需要配置property标签 name属性：set的方法名，去掉set，剩下的单词首字母变小写 ref：指定要注入的bean的id

构造注入

通过调用构造方法注入对象

需要配置constructor-arg index：指定参数下标，从0开始 ref：指定注入的bean的id

注入

整体添加依赖

    <dependencies>
        <!--        spring核心-->
        <dependency>
            <groupId>org.springframework</groupId>
            <artifactId>spring-context</artifactId>
            <version>5.2.1.RELEASE</version>
        </dependency>
        <!--        测试框架-->
        <dependency>
            <groupId>junit</groupId>
            <artifactId>junit</artifactId>
            <version>4.11</version>
            <scope>test</scope>
        </dependency>
    </dependencies>

注入外部bean

使用ref属性注入的就是注入外部bean

定义了两个类

public class ClassA {
    private Person person;
    private String name;
}

public class Person {
    private String name;
}

在配置文件中配置beans-inject.xml

    <bean id="class1" class="com.mwdx.ClassA">
        <property name="name" value="A班"/>
        <property name="person" ref="person1"/>
    </bean>

测试

@Test
public void setBeanTest() {
    System.out.println("测试注入外部bean");
    ApplicationContext context = new ClassPathXmlApplicationContext("beans-inject.xml");
    ClassA class1 = context.getBean("class1", ClassA.class);
    System.out.println(class1.getPerson());
}

注入内部bean

在property中嵌套bean

    <bean id="class2" class="com.mwdx.ClassA">
        <property name="person">
            <bean class="com.mwdx.Person">
                <property name="name" value="111"/>
            </bean>
        </property>
        <property name="name" value="2班"/>
    </bean>

测试

    @Test
    public void setBeanTest2() {
        System.out.println("测试注入内部bean");
        ApplicationContext context = new ClassPathXmlApplicationContext("beans-inject.xml");
        ClassA class2 = context.getBean("class2", ClassA.class);
        System.out.println(class2);
    }

注入普通类型

int、float、double、String、enum 在property中使用value属性

定义实体类

public class User {
    private Integer id;
    private Boolean sex;
    private Double score;
    private String name;
}

配置文件

<!--    注入普通类型-->
<bean id="user1" class="com.mwdx.User">
    <property name="name" value="张三"/>
    <property name="id" value="1"/>
    <property name="sex" value="1"/>
    <property name="score" value="12.12"/>
</bean>

注入数组

    <bean id="user2" class="com.mwdx.User">
        <property name="nums">
            <array>
                <value>11</value>
                <value>22</value>
                <value>33</value>
                <value>44</value>
            </array>
        </property>
    </bean>

p命名空间注入

底层还是set注入

1、配置文件中添加：xmlns:p=“http://www.springframework.org/schema/p"

2、使用

  <bean id="person1" class="com.mwdx.Person" p:name="张三"/>

c命名空间注入

底层是构造器注入

1、配置文件中添加：xmlns:c=“http://www.springframework.org/schema/c"

2、使用

<bean id="person2" class="com.mwdx.Person" c:name="李四"/>

util命名空间，让配置复用

1、配置文件中添加：xmlns:util=“http://www.springframework.org/schema/util"

2、引入约束：http://www.springframework.org/schema/util http://www.springframework.org/schema/util/spring-util.xsd

<!--    util 可复用-->
    <util:list id="util1">
        <value>11</value>
        <value>22</value>
        <value>33</value>
    </util:list>

    <bean id="myCollection" class="com.mwdx.MyCollections">
        <property name="list" ref="util1"/>
    </bean>

自动装配、注入

底层也是通过set注入

Spring可以完成自动化注入，根据名称或根据类自动注入 byName, byType

    <bean id="myCollection1" class="com.mwdx.MyCollections" autowire="byType"/>

引入外部配置文件

注入bean的文件在外部的文件中

1、引入context命名空间，xmlns:context=“http://www.springframework.org/schema/context"

2、引入约束：http://www.springframework.org/schema/context http://www.springframework.org/schema/context/spring-context.xsd

3、使用，${xxx}

    <!--    引入外部配置文件-->
    <context:property-placeholder location="mysql.properies" />
    <bean id="mysql" class="com.mwdx.Mysql">
        <property name="url" value="${jdbc.url}"/>
        <property name="password" value="${jdbc.password}"/>
        <property name="username" value="${jdbc.username}"/>
    </bean>
