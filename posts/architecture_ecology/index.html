Spring Cloud

  • 注册中心：Eureka
  • 复杂均衡：Ribbon
  • 服务通信：Feign
  • Api服务网关：Zuul
  • 配置中心：Spring Cloud Config
  • 集中式日志管理：ELK
  • 分布式链路追踪：
  • 服务保护：Hystrix

Spring Cloud Alibaba

  • 注册中心：Nacos
  • 复杂均衡：Nacos
  • 服务通信：Feign
  • Api服务网关：Gateway
  • 配置中心：Nacos
  • 集中式日志管理：LOG
  • 分布式链路追踪：zipkin Server
  • 服务保护：Sentinel
  • 消息队列：RocketMQ
  • 分布式事务：Seata
  • 任务调度：SchedulerX
  • 分布式存储：OSS
